# 🌌 Mars Mission Control

**Mars Mission Control**은 화성 탐사 콘셉트의 시뮬레이션 대시보드 및 인터랙티브 웹 UI입니다. 동적인 궤도 트래킹, 행성 도달 비주얼(달, 화성 등), 실시간 미션 텔레메트리 데이터를 제공합니다.
**Mars Mission Control** is a sci-fi inspired dashboard and interactivity layer simulating an expedition to Mars. Complete with dynamic trajectory tracking, celestial body approaching visuals (Moon & Mars), and real-time mission telemetry.

## 🚀 주요 기능 (Features)
- **몰입입감 넘치는 SF UI (Immersive Sci-Fi UI)**: 깊은 딥블랙, 네온 액센트, 헤드업 디스플레이(HUD) 디자인.
- **다이내믹 우주 비행 (Dynamic Space Flight)**: 우주선이 접근함에 따라 점점 커지는 실시간 행성 렌더링.
- **미션 텔레메트리 (Mission Telemetry)**: 장애물 관리, 모래시계/블랙홀 아이템 등 인터랙티브 게임 결합 요소.

## 💻 기술 스택 (Tech Stack)
- **프론트엔드 (Frontend)**: React (TypeScript) + Vite
- **스타일링 (Styling)**: Tailwind CSS + Custom CSS Variables for Sci-Fi Themes
- **배포 (Deployment)**: Vercel

## 🛸 빠른 시작 (Getting Started)

```bash
# Clone the repository
git clone https://github.com/manodist/mars-mission-control.git

# Install dependencies
npm install

# Start the development server
npm run dev
```
