# 🌌 Mars Mission Control

**Mars Mission Control** is a sci-fi inspired dashboard and interactivity layer simulating an expedition to Mars. Complete with dynamic trajectory tracking, celestial body approaching visuals (Moon & Mars), and real-time mission telemetry.

## 🚀 Features
- **Immersive Sci-Fi UI**: Deep blacks, neon accents, and heads-up-display (HUD) aesthetics.
- **Dynamic Space Flight**: Real-time rendering of celestial bodies that grow as the ship approaches.
- **Mission Telemetry**: Interactive obstacle management and hourglass/blackhole mechanics.

## 💻 Tech Stack
- **Frontend**: React (TypeScript) + Vite
- **Styling**: Tailwind CSS + Custom CSS Variables for Sci-Fi Themes
- **Deployment**: Vercel

## 🛸 Getting Started

```bash
# Clone the repository
git clone https://github.com/manodist/mars-mission-control.git

# Install dependencies
npm install

# Start the development server
npm run dev
```
